## 📌 Project Overview

This landing page showcases **Paracha Honey Chargulli**, a pure, raw wild honey sourced from pristine mountain regions. The page presents **three jar sizes** (250g, 500g, 1000g) with dynamic pricing, a WhatsApp ordering system, and multiple trust‑building features to maximize sales.

**Key goals:**
- Create a **million‑dollar brand** appearance
- Drive conversions through **psychological triggers** (scarcity, social proof, urgency)
- Provide a seamless **mobile‑friendly** experience
- Enable direct ordering via **WhatsApp** with pre‑filled details
